# Android TV Banner

You need to create a TV banner image for the Android TV launcher.

## Requirements:
- **Filename**: `tv_banner.png`
- **Location**: Place in this directory (`res/drawable-xhdpi/`)
- **Dimensions**: 320x180 pixels (exact)
- **Format**: PNG with transparency support
- **Content**: App logo/name on dark background

## Design Guidelines:
- Use dark background (matches TV leanback interface)
- App name should be clearly visible
- Keep text large and readable from 10 feet away
- Avoid fine details that won't be visible on TV
- Follow Android TV design guidelines

## Quick Creation:
You can use any image editor (Photoshop, GIMP, Canva, etc.) to create a 320x180px image with:
1. Dark background (e.g., #1a1a1a)
2. App name "Anywhere Music Player" in large white text
3. Optional: Music note icon or album art graphic

Once created, save as `tv_banner.png` in this directory.

## Alternative:
If you have an existing app icon, you can use an online tool to generate a TV banner:
- https://romannurik.github.io/AndroidAssetStudio/
- Select "TV Banner" generator
- Upload your icon
- Download and place here as `tv_banner.png`
